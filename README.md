# 🏖️ Ocean View Resort Management System

A web-based Hotel Management Application developed using Java EE technologies.

## 📌 Overview
Ocean View Resort Management System streamlines hotel operations including guest check-in/check-out, room bookings, staff management and reservation handling through a clean web interface.

## 🚀 Features
- 🔑 Secure Admin Authentication
- 👤 Guest Check-in & Check-out Management
- 📆 Room Reservation & Booking System
- 🧑‍💼 Staff Administration Module

## ⚙️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | JSP, HTML, CSS, JavaScript |
| Backend | Java Servlets (Jakarta EE) |
| Database | MySQL |
| Server | Apache Tomcat v10.1 |
| Testing | JUnit 6 |
| Libraries | MySQL Connector |
| IDE | Eclipse |

## 📂 Project Structure
```
OceanViewResort/
├── controller/    # Servlet Controllers
├── DAO/           # Database Access Layer
├── models/         # models/Model Classes
├── OceanTesting/       # JUnit Test Classes
├── webapp/
│   └── *.jsp           # JSP View Pages
└── WEB-INF/lib/        # JAR Dependencies
```

## 🗃️ Database
- Database: MySQL
- Tables: reception, payment_status, reservation, booking, rooms
